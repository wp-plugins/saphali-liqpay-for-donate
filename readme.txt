== Описание ==

Кнопка для приема пожертвований с помощью LiqPay (v 3.0).


== Установка ==

1. Разархивируйте содержимое zip файла в папку плагинов вашего сайта (wp-content/plugins/), используя вашу любимую FTP программу.
3. Активируйте плагин на странице "Плагины" в панели администратора.
4. Все установка завершена.
5. Если данный плагин вы обновили с версии 1.x.x, то обязательно нужно обновить настройки даннного плагина (платежного шлюза).
